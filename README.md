# 🎮 스타크래프트 강의를 난이도별로 볼 수 있는 강의 서비스 제공 🎮

<b>역할</b>
- 카카오 오븐을 통해 와이어프레임 설계
- Restful API로 게이머, 강의, 게시판 글 데이터로 CRUD 기능 구현 
- Redux를 통해 전역상태 관리 
- Kakao, google 로그인 구현
- Admin 페이지에서 강의, 게이머, 회원정보 CRUD 구현
- React-Quill 에디터를 이용한 게시판 기능, 이미지 업로드 구현
